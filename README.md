<h1 align="center">Electronics Calculator</h1>
<p align="center">A simple calculator for series/parallel resistors</p>
<hr>

## Installation

### Clone the repo:

```
git clone https://github.com/httpsKeygen/Electronics-Calculator.git
```

### Build with `g++`:

```
g++ main.cpp && ./a.out
```

### Execute

#### Windows
```
./a.exe
```
#### Linux and MacOS
```
./a.out
```
